## 説明
たくさんのAPIを使用したyoutube代替ウェブアプリケーションです。

地域制限を回避します。

日本語・英語対応です。

アメリカやフランス・イギリスのトレンドも取得できます。

再生方法が数種類あるのでブロックがある環境、音声だけが必要な場合にも利用できます。

ダウンロード機能を実装しています。
